Multilayer Perceptron Class Explaination

In this document I'll explain the process and idea behind MLP class

Ok so the idea here is to go full math. I could use alot of programming techniques like I could go full Object Oriented or at least I could make class for nodes . But I couldn't help myself not chalenging my math skills.
So everything you'll see here is full math.

First 

     private double[][][] weights
These are the weights of the network. Suppose we have weight[a][b][c]. "a" will represent the layer number, "b" is the neuron number and "c" is the weight number that is connected to that neuron.
So it's kinda like this: double[layer][neuron][weight] weights
