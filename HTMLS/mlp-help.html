Ok, Now let's talk buissness. For any ANN to work, you need an Activation function which it's derivitive could be easily calculatable. There are obviously many choices.
From Gaussian function to sigmoid function and etc. So I leave this to you to decide. If you are happy to use the Sigmoid Function which is my favorite, feel free to use Sig-MLP class
else you need to override the function and it's dervitive as I'll explain.
Both the activation function and it's derivitive must accept an input and calculate an output. So it's 1D to 1D. Pretty simple.
